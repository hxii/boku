brew.yaml

Homebrew runner. Runs brew update, brew upgrade, then iterates over a package list to install. Clean alternative to shell loops.

macositerate
› show yaml
# Run `brew update`, `brew upgrade` and install some packages
version: 0.2.4
author: hxii 
description: Run brew updates
variables:
  packages: [eza, bat, jq]
tasks:
  brew_update:
    run: brew update
  brew_upgrade:
    run: brew upgrade
  brew_packages:
    iterate: packages
    run: brew install {}
daily.yaml

Daily brief. Fetches today's date, pulls unfinished Obsidian tasks, grabs weather, and prints a morning summary. Uses save_output to chain variables between tasks.

variablessave_outputobsidian
› show yaml
version: 0.2.4
variables:
  vault: ~/Documents/Brain/
  daily: ${vault}Daily/
tasks:
  date:
    run: date +%Y-%m-%d
    save_output: date
    suppress_output: true
  daily_obsidian:
    if: test -f ${daily}${date}.md
    run: cat ${daily}${date}.md | grep -E '^- \[ \]'
    save_output: daily_tasks
    suppress_output: true
  weather:
    run: curl -s wttr.in/new-york?0T | base64
    save_output: weather
    suppress_output: true
  day:
    run: |
      echo "## Weather"
      echo "${weather}" | base64 -d
      echo ""
      echo "## Today's tasks"
      echo "${daily_tasks}"
pylint.yaml

Linting pipeline. Checks that ruff exists, then runs linting and formatting in order with depends_on. Shows on_failure handling.

depends_onon_failurepython
› show yaml
version: 0.2.4
author: Paul (hxii) Glushak
description: Run ruff code linting and formatting

tasks:
  ensure_ruff:
    run: command -v ruff
    on_failure: echo "Ruff not available!"
  ruff_lint:
    depends_on: [ensure_ruff]
    run: ruff check . --fix
  ruff_format:
    depends_on: [ruff_lint]
    run: ruff format .
  confetti:
    depends_on: [ruff_lint, ruff_format]
    run: echo "All good!"
setup.yaml

System bootstrap. Installs Homebrew if missing, then installs dotter and clones your dotfiles. Uses if and depends_on to orchestrate flow.

ifdepends_onbootstrap
› show yaml
# System setup using brew and dotter
version: 0.2.4
author: hxii 
description: System Configurator
variables:
  dots: "git.sr.ht:~hxii/dotfiles"
  brew: /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
tasks:
  install_brew:
    if: "! command -v brew"
    run: ${brew}
  ensure_brew:
    run: command -v brew
  install_dotter:
    if: "! brew list -1 | grep dotter"
    depends_on: [ensure_brew]
    run: brew install dotter
  clone_dos:
    working_dir: ~/
    run: git clone ${dots}
  dotter:
    working_dir: ~/dotfiles
    run: dotter
system-health.yaml

Health check. Disk usage, memory info, active users, temp file cleanup with iterate, and a final report generator. Works on macOS and Linux.

iterateplatformreporting
› show yaml
version: 0.2.4
author: Paul (hxii) Glushak 
description: System health check (macOS + Linux)

variables:
  disk_warn_pct: 80
  clean_dirs:
    - /tmp
    - ~/.Trash
  report_file: /tmp/system-report-${env:USER}.txt

tasks:
  hostname:
    description: Show the system hostname
    run: hostname -s

  disk_check:
    description: Check root disk usage
    run: df -h / | tail -1
    save_output: disk_line

  memory_usage:
    description: Show memory info
    run: |
      if [ "$(uname)" = Darwin ]; then
        vm_stat | head -3
      else
        free -h
      fi

  cleanup_preview:
    description: Preview cleanup targets
    depends_on: [disk_check]
    run: |
      for d in /tmp ~/.Trash; do
        t=$(eval echo "$d")
        test -d "$t" && du -sh "$t" 2>/dev/null
      done
    save_output: cleanup_info

  do_cleanup:
    description: Clean temp and trash dirs
    depends_on: [cleanup_preview]
    if: test -d /tmp
    run: 'rm -rf {} 2>/dev/null; echo Cleaned'
    iterate: clean_dirs

  generate_report:
    description: Write health report to file
    depends_on: [hostname, disk_check, memory_usage]
    run: |
      {
        echo "=== System Health ==="
        echo "Host: $(hostname -s)"
        echo "User: ${env:USER}"
        echo "Date: $(date)"
        echo ""
        df -h /
      } | tee "${report_file}"

  quick:
    description: Quick system overview (no deps)
    run: |
      echo "Host: $(hostname -s)"
      echo "OS:   $(uname -srm)"
      echo "Up:   $(uptime | awk -F 'up ' '{print $2}' | awk -F ',' '{print $1}')"
      echo "Disk: $(df -h / | tail -1 | awk '{print $3}') used
